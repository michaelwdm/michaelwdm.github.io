我的态度

代码要写得干净，问题要追得彻底，东西要做得漂亮。一个 bug 不解决就睡不着，一个功能没完善就不想发布。我相信细节决定成败——每一行代码都值得认真对待，每一个设计选择都应该有它的理由。

编程不只是一种技能，它教我把大问题拆成小问题，把模糊的需求变成清晰的步骤，把重复的工作交给机器。写代码和写文章本质上是同一件事：用逻辑和结构来表达思考。而我想把这两件事都做好。

一个真实的小故事：有一回为了爬一个网站的数据，我花了整整一个周末研究它的请求头参数——试了十几种组合，失败了好多次，终于在周日晚上拿到了想要的数据。那一刻的成就感，比打游戏通关还要爽十倍。我觉得这就是编程的魅力：它一直在奖励那些不肯放弃的人。

学习编程就像学一门新的语言——一开始磕磕绊绊，慢慢能说完整的话，最终能用它来表达复杂的思想。我现在大概处于"能聊天但偶尔还会犯语法错误"的阶段，但每一天都比昨天更流利一点。

📍 基础

🐍 Python 基础

从最基础的 print("Hello World") 开始，到变量与数据类型、流程控制、函数定义、面向对象编程、异常处理、文件操作、模块与包的管理。每一个概念都反复练习，每一行代码都自己敲过。现在能独立写出几百行的实用脚本，回头看，当初觉得很难的东西现在已经变成肌肉记忆了。

📍 实战

🕸️ 爬虫实战

HTTP 协议基础、requests 库发起请求、BeautifulSoup 解析网页、XPath 精准定位元素。学会了用 User-Agent 伪装和请求频率控制来应对反爬机制，也在研究动态加载页面的应对策略。过程中踩过无数坑——被封过 IP、被返回过空白页、被验证码拦住——但每一次都逼着我学新东西。从只会看教程到能自己独立完成一个完整的爬虫任务，这条路走得很有成就感。

📍 分析

📊 数据分析

pandas 进行数据清洗和转换，matplotlib 绘制折线图、柱状图、散点图，numpy 进行数值计算，还有基础统计分析。把一堆杂乱无章的原始数据，经过清洗、整理、可视化，最终变成一目了然的图表和结论——这个过程让我第一次感受到"数据的力量"。数据不会说谎，但你得学会怎么跟它对话。

📍 探索

🤖 AI 入门

机器学习的基本概念，监督学习与无监督学习的区别，线性回归和决策树的原理，神经网络的基础结构。正在初步接触 TensorFlow 和 PyTorch。虽然还处在"门槛上张望"的阶段，但已经能感受到这个领域的广阔和迷人。AI 不只是写代码，它需要数学、直觉和耐心的三重加持——这正是它吸引我的地方。

📍 根基

🧮 算法与数据结构

数组、链表、栈、队列、二叉树、排序算法、查找算法……这些是计算机科学的根基，也是我目前花时间最多的学习内容。正在啃《算法导论》——进度不快，但每道题都要求自己先独立思考，再去看参考答案。我知道这条路没有捷径，唯一的方法就是一道题一道题地啃，一个概念一个概念地消化。

我的思考

不只是写代码，也想清楚为什么要写

💻

关于编程

编程对我来说不只是工具，更是一种思维方式。它教会我把大问题拆成小问题，把模糊的需求变成清晰的步骤，把重复的工作交给机器。每次写出一段高效优雅的代码，都像完成了一件精心打磨的艺术品。代码是写给人看的，顺便能在机器上运行——这句话我一直记在心里。

🧠

关于 AI

人工智能是当下最激动人心的技术方向。我关注它的发展，也思考它的边界。AI 能帮我们做什么？不能做什么？它将如何改变我们的学习方式、工作方式和思考方式？作为一个学习者，我想掌握这项技术；作为一个思考者，我也想保持清醒的判断。技术永远只是手段，真正重要的是我们用它来做什么。

📚

关于学习

我不觉得学习是苦差事。解决一个难题的快乐、搞懂一个概念的快乐、做出一个小作品的快乐——这些都是真实的正反馈。当然会有卡壳的时候，会有看了三遍还不懂的时候，会有运行报错却找不到原因的时候。但睡一觉起来继续啃，换一个角度重新想，或者去 Stack Overflow 上搜一搜，总能找到出路。学习不是比谁快，而是比谁走得远。

🎯

关于未来

我想考一所好大学的计算机专业，深入学习人工智能，将来从事技术研发。路还很长，但方向已经明确，剩下的就是一步一步走。不急躁、不放弃、不敷衍——认真对待每一行代码、每一个概念、每一次尝试。我相信持续积累的力量，也相信热爱的力量。做自己喜欢的事，本身就赢了一半。

阅读与写作

除了技术，文字也是我的热爱

📚

📖 我的阅读

阅读是我学习技术、理解世界的重要方式。我读书不追求数量，更在意有没有真正消化。技术书籍帮我打牢基础，科幻小说打开我的想象力，科普读物让我保持对世界的好奇，人物传记则让我看到不同人生的可能性。

我喜欢的阅读类型：

🐍 Python 编程 🧠 深度学习与AI 🧮 算法经典 🚀 硬科幻小说 🔬 科普读物 📜 人物传记 💡 思维方法

📌 目前在啃一本深度学习的经典著作，进度艰难但非常充实。喜欢刘慈欣笔下那种跨越光年的宏大叙事，也喜欢费曼那种把复杂问题讲得通俗易懂的有趣灵魂。

✍️ 我的写作

遇到有意思的想法就随手记下来，学习中有了心得体会就整理成文章，碰到搞懂了一个难点的时刻就写一篇笔记来巩固理解。写作帮我梳理思路——很多时候，你以为自己懂了，直到试着写出来才发现还有很多模糊的地方。写作强迫我把事情想清楚、讲明白。

编程和写作，一个用代码表达逻辑，一个用文字表达思考。两者看似不同，本质相通——都是在用结构化的方式传达信息。我相信，把这两件事都做好，能让我成为一个更完整的创作者。
